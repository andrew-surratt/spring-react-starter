# React Starter

This template provides a starter for React projects.

## Features

- React
- Vite
- Typescript
- Auth0 Authentication
- Prettier
- ESLint
- Automated formatting using Husky hooks

## Install

`npm install`

## Start

`npm run dev`

## Build

`npm run build`

## Lint

`npm run lint`

## Prettier

`npm run prettier`
